= My dot files

== Prerequisites

* A working RVM installation

  chmod +x ~/.rvm/hooks/after_cd_bundler

== Installation

  chsh -s /bin/zsh

  git clone git://github.com/der-flo/dotfiles ~/dotfiles
  cd ~/dotfiles
  git submodule init
  git submodule update
  rake install
