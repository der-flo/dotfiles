= My dot files

== Prerequisites

* A working RVM installation
* A terminal with support for 256 colors (<tt>TERM=xterm-256color</tt>)

== Installation

  brew install fish ripgrep
  chsh -s /usr/local/bin/fish
  git clone git://github.com/der-flo/dotfiles ~/dotfiles
  cd ~/dotfiles
  rake install
